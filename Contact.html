PCB Assembly Tools

About This Project

Project Overview

PCB Assembly Tools is a comprehensive suite of client-side web applications designed specifically for electronics engineers, PCB assembly specialists, and component procurement professionals. All tools are built with privacy and performance in mind, running entirely in your browser without requiring any server infrastructure.

This project integrates seamlessly with industry-leading supplier APIs including Digikey and Mouser, providing real-time access to component data, specifications, pricing, and availability. All API credentials and user data are stored locally in your browser using modern client-side storage technologies, ensuring your information never leaves your device.

Key Features

  • Excel-based BOM processing with intelligent column mapping
  • Real-time component scanning using QR codes and barcodes
  • Automated part matching and alternate component suggestions
  • Multi-supplier price comparison and optimization
  • Inventory tracking with stock alerts and reorder automation
  • Client-side data storage with no server dependencies
  • Offline-capable with cached component data
  • Professional-grade reporting and export capabilities

Technology Stack

Built using pure HTML5, CSS3, and vanilla JavaScript with a modular architecture. The project follows Swiss minimalist design principles with a monochrome aesthetic using the Geist font family. All code is organized for maximum reusability and maintainability.

API Integration

This project utilizes official APIs from major electronic component distributors. To use the full functionality of these tools, you'll need to register for API access:

Digikey API

OAuth 2.0 authentication with client ID and client secret. Supports product search, ordering, and barcode lookup.

Register for Digikey API

Mouser API

Simple API key authentication. Separate keys for search and cart/order operations.

Register for Mouser API

Once registered, configure your API credentials in the settings panel (gear icon in the navigation bar). All credentials are stored locally in your browser and never transmitted to any third-party servers.
