Active Guards
TRUNCATION_DETECTOR AUTOMATED
Scanner flags: "[...]", "...", "and so on", "etc.", "similar entries follow", "you can expand this", "[rest of X]", "[continue pattern]". Each hit: −0.25.
PLACEHOLDER_DETECTOR AUTOMATED
Flags: "[TBD]", "[insert X]", "[add more here]", "[details TBD]", "TODO", empty bullet points, headers with no following body. Each hit: −0.20.
SECTION_DENSITY_GATE HARD
Each H2/H3 section scored independently. Section below floor: −0.15. Output is NOT complete until all sections clear their floor. There is no partial credit for a section that stops early.
LENGTH_SCORE_GATE HARD
length_score = min(actual_loc / 350, 1.0). This is the highest-weight metric (0.40). A document that stops at 150 LOC scores 0.43 on this dimension. That kills the final score.
CROSS_SECTION_CONSISTENCY ON
Facts, names, rules, and relationships stated in one section must be consistent across all others. Each contradiction: −0.08. The critic model scans all cross-references.
EARLY_STOP_DETECTION AUTOMATED
"I've outlined the key sections" / "you can expand from here" / "this gives you a foundation" = EARLY STOP. Detected the same as truncation. The oracle does not grade foundations. It grades completed documents.
