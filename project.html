Overview

SalesforceCore is a comprehensive .NET library designed to bridge the gap between complex enterprise requirements and the Salesforce ecosystem. It provides a robust, type-safe, and highly performant layer for building scalable integrations.

The framework was built to solve the common pain points of Salesforce integration: loose typing, complex authentication flows, and lack of maintainable query structures.

Key Features
  • Strongly-typed data access with compile-time checking
  • LINQ-to-SOQL query provider for intuitive querying
  • Multiple authentication flows (PKCE, JWT, Client Credentials)
  • High-volume operations via Bulk API 2.0 and Composite APIs
  • Dynamic UI generation based on Salesforce metadata
  • ASP.NET Core integration with Tag Helpers
Architecture

The library is designed with a modular architecture to support various implementation scenarios, from lightweight background workers to full-scale web applications.
